# ⚡ Household Electric Power Consumption Dashboard

## Course
Exploratory Data Analysis | Instructor: Ali Hassan Sherazi

## How to Install
pip install -r requirements.txt

## How to Run
streamlit run app.py

## Charts Included
1. Histogram - Active power distribution
2. Line Chart - Monthly trends
3. Bar Chart - Hourly patterns
4. Pie Chart - Energy by season
5. Scatter Plot - Power vs intensity
6. Box Plot - Spread by weekday
7. Heatmap - Feature correlations
8. Area Chart - Sub-metering trends
9. Count Plot - Records per year
10. Violin Plot - Density by season

## Key Insights
- Winter uses 31% of total energy (highest - due to heating)
- Summer uses only 17% (lowest)
- Peak usage hours are 7-9 AM and 7-9 PM
- Strong correlation between active power and intensity (0.99)

## Dataset
Source: UCI Machine Learning Repository
File: household_power_consumption.txt
Period: December 2006 - November 2010
Records: 2,075,259 rows
